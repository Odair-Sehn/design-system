# Theming

The `ds-ui` package offers the default DS theme set by the design system and its style guide. It's a series of CSS variables that are used globally by all components.

To use the default theme, you should import it in your main application entrypoint (with the required Dass Symbols styles):

```js
// my main app script
import "ds-ui/dist/global.css"; // the global required files used by the DS
import "ds-ui/dist/dass-symbols.css"; // the icon font, required by the theme
import "ds-ui/dist/dass-tokens.css"; // the default DS theme
```

If you want to replace these values with your app style guide, changing colors, font sizes, spacing and other appearance settings, you can create a new CSS file for your application and import it in place of the default theme:

```js
// my main app script
import "ds-ui/dist/global.css"; // the global required files used by the DS
import "my-custom-theme.css"; // your new, shiny theme
```
