# Tematização

O pacote `dass-ds-ui` oferece o tema padrão definido pelo sistema de design e seu guia de estilo. É uma série de variáveis CSS que são usadas globalmente por todos os componentes.

Para usar o tema padrão, você deve importá-lo em seu arquivo de inicialização do aplicativo:

```js
// main.js ou seu arquivo de inicialização do app
import "dass-ds-ui/dist/dass-style.css"; // tokens do dass design system
```

Se quiser substituir esses valores pelo guia de estilo do aplicativo, alterando cores, tamanhos de fonte, espaçamento e outras configurações de aparência, você pode criar um novo arquivo CSS para seu aplicativo e importá-lo no lugar do tema padrão:

```js
// main.js ou seu arquivo de inicialização do app
import "my-custom-theme.css"; // seu arquivo de tema customizado
```

Você pode conferir uma explicação para todos os tokens de estilo e como eles são usados na seção Guia de estilo desta documentação. O conjunto completo de variáveis CSS dos tokens segue para referência:
