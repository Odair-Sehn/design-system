# Sombras

Os seguintes tokens controlam a aparência da sombra em seu tema. Você pode criar novos temas definindo variáveis CSS com valores diferentes, e o dass design system substituirá seus valores padrão pelas sombras do tema de acordo.

Por exemplo, se você deseja criar um novo tema com uma nova sombra pequena:

```css
/* my-custom-theme.css */

:root {
  --dass-shadow-sm: 0px 4px 7px 0px; /* novos tamanhos de sombra personalizados */
}

.your-custom-element {
  box-shadow: var(--dass-shadow-sm);
}
```

The shadow tokens need to be used together with a color token to set the shadow color.

| Token            | Preview                                             |
| ---------------- | --------------------------------------------------- |
| `dass-shadow-sm` | <span class='dass-sb-shadow dass-shadow-sm'></span> |
| `dass-shadow-md` | <span class='dass-sb-shadow dass-shadow-md'></span> |
| `dass-shadow-lg` | <span class='dass-sb-shadow dass-shadow-lg'></span> |

# Contorno do Foco

O contorno do foco é um valor CSS `outline` usado para aplicar uma sombra sutil ao elemento em foco para uso de acessibilidade.

```css
.your-form-control:focus {
  outline: var(--dass-shadow-focus);
}
```

| Token                 | Preview                                                |
| --------------------- | ------------------------------------------------------ |
| `--dass-shadow-focus` | <span class='dass-sb-shadow dass-shadow-focus'></span> |
