# Tipografia

Os tokens a seguir controlam a tipografia das fontes em seu tema. Você pode usá-lo como variáveis CSS em seus estilos de aplicativo para herdar os valores padrão em um elemento personalizado:

```css
/* my-custom-element.css */
.my-custom-element {
  font: var(--dass-font-body--default__regular);
}
```

Você também pode criar um novo arquivo de tema CSS e personalizar esses valores com os valores do guia de estilo do aplicativo.

```css
/* my-custom-theme.css */

:root {
  --dass-font-body--default__regular: 500 1.5rem/1.65 var(--font-family--base);
}
```

## Tokens Básicos

Os tokens a seguir definem o estilo e tamanho básico da fonte para todos os elementos.

| Token                   | Default Value                                                             |
| ----------------------- | ------------------------------------------------------------------------- |
| `--font-family--base`   | "Roboto", "Helvetica Neue", Helvetica, "Nimbus Sans L", Arial, sans-serif |
| `--font-family--system` | "Inter", system-ui, sans-serif                                            |
| `--font-family--icons`  | "Dass Symbols"                                                            |
| `--base-font-size`      | 16px                                                                      |
