# Typography

The following tokens control the typographic and fonts appearance in your theme. You can use it as CSS variables in your application styles to inherit the default values in a custom element:

```css
/* my-custom-element.css */
.my-custom-element {
  font: var(--font-body--default__regular);
}
```

You can also create a new CSS theme file and customize these values with your app style guide values.

```css
/* my-custom-theme.css */

:root {
  --font-body--default__regular: 500 1.5rem/1.65 var(--font-family--base);
}
```

## Basic Tokens

The following tokens set the DS font stacks and basic font size for all elements.

| Token                   | Default Value                                                             |
| ----------------------- | ------------------------------------------------------------------------- |
| `--font-family--base`   | "Roboto", "Helvetica Neue", Helvetica, "Nimbus Sans L", Arial, sans-serif |
| `--font-family--system` | "Inter", system-ui, sans-serif                                            |
| `--font-family--icons`  | "Dass Symbols"                                                            |
| `--base-font-size`      | 16px                                                                      |
