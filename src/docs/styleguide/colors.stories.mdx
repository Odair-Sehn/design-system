# Cores

Os tokens a seguir controlam a aparência da cor em seu tema. Você pode usá-lo como variáveis CSS em seus estilos de aplicativo para herdar os valores padrão em um elemento personalizado:

```css
/* my-custom-element.css */
.my-custom-element {
  background-color: var(--dass-color--primary);
}
```

Você também pode criar um novo arquivo de tema CSS e personalizar esses valores com os valores do guia de estilo do aplicativo.

```css
/* my-custom-theme.css */

:root {
  --dass-color--primary: #CB3645;
}
```

Agora o Design System usará sua nova cor primária em seu aplicativo.
