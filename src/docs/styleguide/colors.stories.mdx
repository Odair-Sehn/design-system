# Colors

The following tokens control the color appearance in your theme. You can use it as CSS variables in your application styles to inherit the default values in a custom element:

```css
/* my-custom-element.css */
.my-custom-element {
  background-color: var(--dass-color--primary);
}
```

You can also create a new CSS theme file and customize these values with your app style guide values.

```css
/* my-custom-theme.css */

:root {
  --dass-color--primary: #CB3645;
}
```

Now the Design System will use your new primary color in your app.
