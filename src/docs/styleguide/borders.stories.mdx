# Borda

Os tokens a seguir controlam a aparência da borda em seu tema. Você pode usá-lo como variáveis CSS em seus estilos de aplicativo para herdar os valores padrão em um elemento personalizado:

```css
/* my-custom-element.css */
.my-custom-element {
  border-radius: var(--dass-border-radius-sm);
}
```

Você também pode criar um novo arquivo de tema CSS e personalizar esses valores com os valores do guia de estilo do aplicativo.

```css
/* my-custom-theme.css */

:root {
  --dass-border-radius-sm: 3px;
}
```

## Raio

Os seguintes tokens controlam o tamanho do raio da borda dos elementos:

| Token                     | Value | Preview                                                    |
| ------------------------- | ----- | ---------------------------------------------------------- |
| `--dass-border-radius-sm` | `2px` | <span class='dass-sb-border dass-border-radius-sm'></span> |
| `--dass-border-radius-md` | `4px` | <span class='dass-sb-border dass-border-radius-md'></span> |
| `--dass-border-radius-lg` | `8px` | <span class='dass-sb-border dass-border-radius-lg'></span> |
