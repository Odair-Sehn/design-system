# Borders

The following tokens control the border appearance in your theme. You can use it as CSS variables in your application styles to inherit the default values in a custom element:

```css
/* my-custom-element.css */
.my-custom-element {
  border-radius: var(--border-radius-sm);
}
```

You can also create a new CSS theme file and customize these values with your app style guide values.

```css
/* my-custom-theme.css */

:root {
  --border-radius-sm: 3px;
}
```

## Radius

The following tokens control the border radius size of elements:

| Token                | Value | Preview                                               |
| -------------------- | ----- | ----------------------------------------------------- |
| `--border-radius-sm` | `2px` | <span class='dass-sb-border border-radius-sm'></span> |
| `--border-radius-md` | `4px` | <span class='dass-sb-border border-radius-md'></span> |
| `--border-radius-lg` | `8px` | <span class='dass-sb-border border-radius-lg'></span> |
